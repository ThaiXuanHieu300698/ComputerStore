# ComputerStore
 
## How to Install
1. Open ComputerStore.sln
2. Open Folder DATA in DAL
3. File DBComputerStore.sql
4. Open SqlServer
5. Open flie DBComputerStore.sql
6. Press F5
7. Click Start on Visual Studio
